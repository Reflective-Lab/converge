```
Morning:    focus -> sync -> next
Work:       fix, check, pr

Developer:
dev            Start local dev environment

Product:

Engineering:
audit          Security, compliance, drift review
review <pr>    Review a pull request

Ops:
sync           Team activity and build health
deploy         Deploy to production
```
