# EXP-001: Validate mathematical suggestors against published reference data

**Date:** 2026-04-21
**Status:** confirmed
**Hypothesis:** Every converge suggestor that implements a named mathematical algorithm produces results matching independently published reference answers for the same inputs.
**Falsification criterion:** Any suggestor producing a result that diverges from the known correct answer by more than floating-point tolerance (1e-6 for exact algorithms, 10% for heuristics) falsifies the hypothesis for that suggestor.
**Method:** Build a reference validation test suite using textbook/published inputs with known correct outputs. Run converge solvers against these inputs and compare.

## Scope

### Exact algorithms (must match within f64 epsilon)

| Algorithm | Suggestor | Reference Source |
|-----------|-----------|-----------------|
| Hungarian (O(n³)) | assignment_pack | Kuhn 1955 — textbook 3×3 and 4×4 cost matrices with known optimal |
| Hopcroft-Karp bipartite matching | formation_assembly, provider_selection | Hopcroft & Karp 1973 — standard examples from graph theory |
| 0-1 Knapsack DP | knapsack module | Martello & Toth "Knapsack Problems" — small instances with known optima |
| EOQ formula | inventory_replenishment | Harris 1913 / Wilson formula — EOQ = √(2DS/H) |
| Min-cost flow | network_flow | Ahuja, Magnanti, Orlin — small network examples |
| Dijkstra shortest path | graph::dijkstra | Dijkstra 1959 — textbook weighted graph |

### Statistical methods (must match within f64 epsilon)

| Algorithm | Suggestor | Reference Source |
|-----------|-----------|-----------------|
| Z-score | anomaly_detection, anomaly_triage | Hand-computable: mean, stddev, z = (x-μ)/σ |
| Descriptive statistics | descriptive_stats | NIST Statistical Reference Datasets (StRD) — Lew dataset |
| Logistic regression (inference) | classification | Scikit-learn reference: known weights + bias + sigmoid |
| Linear regression (inference) | regression | Trivial: y = w·x + b |
| Cosine similarity | similarity | Hand-computable: cos(θ) = (A·B)/(‖A‖·‖B‖) |
| Exponential smoothing | forecasting | Hyndman & Athanasopoulos "FPP3" — SES worked example |

### Heuristics (must be within known approximation bounds)

| Algorithm | Suggestor | Reference Source |
|-----------|-----------|-----------------|
| Greedy set cover | setcover | Chvátal 1979 — O(ln n) approximation guarantee |
| First-fit decreasing bin packing | bin_packing | FFD guarantee: ≤ (11/9)OPT + 6/9 |
| K-means clustering | segmentation | Well-separated clusters must converge to correct centroids |
| Nearest-neighbor + 2-opt TSP | traveling_salesman | TSPLIB att48 (48-city, known optimal = 10628) — heuristic should be within 15% |
| WSJF scoring | backlog_prioritization | SAFe WSJF formula: (Business Value + Time Criticality + Risk Reduction) / Job Duration |

### K-Means note

K-means with random initialization can converge to local minima even with well-separated clusters. The 4-cluster test uses multi-seed retry (up to 20 seeds) to account for this. The 2-cluster test with seed=42 converges reliably due to the simpler structure.
